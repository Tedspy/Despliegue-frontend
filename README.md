# Despliegue Frontend

Este es el frontend del proyecto de Fútbol Colombiano. Es una aplicación web estática construida con **HTML5, CSS3 y Vanilla JavaScript**.

## Uso
Como es un sitio estático, no requiere instalación de paquetes de Node.js.
Para visualizar el proyecto:
1. Clona el repositorio.
2. Abre el archivo `index.html` en tu navegador de preferencia, o utiliza una extensión como **Live Server** en VS Code para tener recarga automática.

El frontend consume la API ubicada en `http://localhost:3000/api/equipos` (ver repositorio del backend).
